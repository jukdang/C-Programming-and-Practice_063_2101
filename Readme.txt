합 100판 이상을 해봤는데, 일반모드는 레벨 7이되면 눈으로 따라가기 힘든 수준, max level로 정했고, 5000점을
넘지못하고, 플레이타임은 7레벨까지 간다고 산정했을때 5~7분사이

챌린지의 경우 일반모드와 속도증가하는 주기를 다르게 선정, 플레이해보면서 밸런스를 맞추었음.
저희 개발자들과, 주변 지인에게 테스트?를 맡겨 플레이 해본결과, 실력자 기준 3판정도에 1번 클리어,
일반인 기준 10이상에 1번 클리어할까말까 하는 난이도.
초반에는 층수를 낮추는게 생각보다 쉽다고 느끼지만 속도가 올라가면서 조금씩 힘들어지고 블럭이 적을때
최소층으로 잘 쌓는게 생각보다 어렵다고 플레이어들이 말했음.
체감난이도는 낮지만 생각보다 클리어는 어려워서 도전정신, 중독성을 이끌어 낸다고 생각함.
